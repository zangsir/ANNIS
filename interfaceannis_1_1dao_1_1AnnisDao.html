Properties annis.dao.AnnisDao.getCorpusConfiguration ( String  corpusName)

Get a specific configuration of a corpus from directory.

The corpus config files are actually stored in the

<user>/.annis/data/corpus

directory, decorated with a UUID. The actual name of a specific corpus property file is stored in the media_file table.

Returns
The corpus configuration is represented as Key-Value-Pairs.
