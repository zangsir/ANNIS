String annis.gui.controlpanel.SearchOptionsPanel.checkSegments ( String  key,
Set< String >  corpora 
)
private

Checks, if all selected corpora have the same default segmentation layer.

If not the tok layer is taken, because every corpus has this one.

Parameters
keythe key for the segementation config, must be KEY_DEFAULT_BASE_TEXT_SEGMENTATION or KEY_DEFAULT_CONTEXT_SEGMENTATION.
corporathe corpora which has to be checked.
Returns
"tok" or a segment which is defined in all corpora.

If no segment is set in the corpus config use always the tok segment.
